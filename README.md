# RxJS Operators Playground

An interactive playground for exploring RxJS operators with live demos, marble diagrams, and categorized navigation.

## Features

- **Live operator demos** — run and observe RxJS operators in real time
- **Marble diagrams** — visual representations of operator behavior
- **Categorized browsing** — operators organized by category:
  - Creation
  - Transformation
  - Filtering
  - Combination
  - Error Handling
  - Multicasting
  - Utility
- **Interactive controls** — some operators include interactive elements
- **Deep linking** — share links to specific operators via URL hash

## Tech Stack

- TypeScript
- RxJS 7
- Vite

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
